⚝ How To use? ⚝

Command Description
--w Fetch results from web search. This command will search the web for the given query and return a list of links to relevant websites.
--i Improve text - fix grammar. This command will improve the grammar of the given text. It will correct spelling errors, punctuation errors, and grammatical errors.
--e Write emails. This command will write an email based on the given instructions. It will include the recipient's address, the subject line, and the body of the email.
--r Rewrite/rephrase text. This command will rewrite or rephrase the given text. It will change the wording of the text to make it more concise, clear, or persuasive.
--b Expand line / make lines bigger. This command will expand the given text to make the lines bigger. This is useful for addinf more content into the text.
--s Summarize content. This command will summarize the given content. It will identify the main points of the content and present them in a concise way.
